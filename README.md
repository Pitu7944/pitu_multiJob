### 🏠 [Homepage](https://github.com/Pitu7944/pitu_multiJob#readme)

## Usage
<b>Setting a Job:</b>
```
/pmj_setjob {id} {job} {grade}
```

<b>Removing a Job:</b>
```
/pmj_unsetjob {id}
```

<b>Checking a Job:</b>
```
/pmj_checkjob {id}
```

<b>Getting info about own Job:</b>
```
/pmjinfo
```
